### Function
Barracks Towers house a number of guards equal to the level

### Levels
1. 1 max guard      80 blocks max Patrol distance
2. 2 max guards    110 blocks max Patrol distance
3. 3 max guards    140 blocks max Patrol distance
4. 4 max guards    170 blocks max Patrol distance
5. 5 max guards    200 blocks max Patrol distance

### Skills
- Archer:
	- Primary: Agility
	- Secondary: Adaptability
- Druid:
	- Primary: Mana
	- Secondary: Focus
- Knight:
	- Primary: Adaptability
	- Secondary: Stamina

### Limits
The max guard limit seems to be enforced through the UI

The max patrol distance hasn't been confirmed yet
